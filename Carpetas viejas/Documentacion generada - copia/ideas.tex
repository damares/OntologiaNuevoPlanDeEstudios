\section{Mencionar nuestras ideas sobre cómo sacar partido a esto}

\todo{Hablar los puntos mencionados por }
\begin{itemize}
\item Se supone que la \emph{Description Logic} permite el
  razonamiento sobre la ontología para así descubrir errores en su
  concepción (TBox) o en sus datos (ABox).
\item Presentación de la información contenida en la ontología para
  diferentes perfiles: profesores, alumnos, secretaría, legos,
  etc. Vía exportación a html o a herramientas visuales (grafos,
  etc.).
\item La ontología es en sí misma un modelo de datos por lo que se
  puede generar a partir de ella diferentes modelos de datos: UML,
  relacional, no-sql, etc. (Daniel tiene algunas cosas apuntadas de un
  antiguo org).
\end{itemize}
